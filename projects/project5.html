Raspberry pi servo controller

Hi, this is my first project that i share with you and im very exited about it! I and ChatGPT have made a program that utilizes the Raspberry pi's GPIO pins to control a servo motor using a virtual slider

The challenges that i encountered

As in every project that i try, python tries to bombard GPT's code with errors. And this project is no exeption, but after half an hour of troubleshooting later i managed to get this code working corectly and to share it with you.

Description

The project uses a micro servo atached to an rpi 5 and controlled by a virtual slider. But the servo jiters a bit if you have sugestions about fixing this issue let me know.

Tutorial

What you need

You will need a raspberry pi (any model), a servo motor, the code and some jumper wires.

Asembly

Open the code here

Change the servo_pin variable with the GPIO pin number you're using to connect the servo.

Adjust the min_duty and max_duty variables based on your servo's specifications. These values represent the minimum and maximum duty cycles corresponding to 0 and 180 degrees respectively.

Conect the servos GND pin to ground, 5V to 5V and the signal pin to whatever pin you chose.

Start the program.

Enjoy
