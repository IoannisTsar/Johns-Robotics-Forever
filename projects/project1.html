The LED project

I'm making a system with only LEDs, resistors, cables, microcontrllers and of course code. I am calling it The LED project.

What is it?

The LED project consists of simple electronic circuits (having as major parts one microcontroller and some LEDs each) wich with code are havig life. The code cleverly uses the restricted parts to make spectacular pieces of art.
